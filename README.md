# Professional Portfolio & Resume – Emerald Slate Edition

A high-performance, responsive, and print-optimized professional resume built with **HTML5**, **Tailwind CSS**, and **Custom CSS3**. This project features a modern "Dark Mode" aesthetic, optimized for both screen viewing and high-quality PDF export.

---

## 🚀 Key Features

* **Emerald-Slate Dark Theme**: A premium, high-contrast design inspired by modern developer tools and SaaS platforms.
* **Print-Optimized (PDF Ready)**: Integrated `@media print` CSS logic that automatically converts the dark-themed web view into a clean, professional, printer-friendly format.
* **Responsive Architecture**: Fully fluid layout using **Tailwind CSS Grid and Flexbox**, ensuring a perfect viewing experience on mobile, tablet, and desktop.
* **Interactive Components**: Hover-responsive project cards with direct links to live deployments and source code.
* **Typography**: Powered by the **Inter** font family for maximum readability across all devices.

---

## 🛠 Tech Stack

* **Core**: HTML5, CSS3
* **Styling**: Tailwind CSS (CDN-based for zero-config portability)
* **Custom Styling**: Specialized CSS for dark-mode depth, emerald accents, and print-layer overrides.
* **Icons**: Unicode-based iconography for fast, lightweight asset loading.

---

## ⚙️ How to Customize
* **Change Colors**: Modify the primary emerald accent (`#10b981`) or the slate background (`#020617`) in the custom classes within `style.css`.
* **Update Data**: Open `index.html` and replace the placeholder text within the `Professional Experience` and `Featured Projects` sections.

---

## 📄 Exporting to PDF

This project is specifically engineered for the "Print to PDF" function:
* Open the `index.html` in any Chromium-based browser (Chrome, Edge, Brave).
Click the **Print / Download PDF** button at the footer.
Ensure **Background Graphics** is turned ON in the print settings if you want the emerald accents to appear in the PDF.
